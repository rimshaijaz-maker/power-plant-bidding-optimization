# Power Plant Bidding Optimization

This project develops an optimization-based bidding framework for a combined cycle gas turbine (CCGT) power plant operating under CAISO-style market conditions.

## Project Overview

The goal of this project is to model how a thermal power plant can structure its bids and operating decisions in an electricity market while accounting for operational and economic constraints.

The analysis focuses on a 2x1 CCGT plant configuration and incorporates key market and plant-level cost components, including:

- Startup cost
- Minimum load cost
- Energy bid construction
- Fuel cost assumptions
- Carbon cost adjustments
- Heat rate-based operating economics

## What This Project Does

This project:

- Translates plant operating characteristics into market bidding logic
- Evaluates cost-based bid construction for different plant configurations
- Applies engineering and market assumptions to support dispatch decisions
- Demonstrates how optimization and structured assumptions can be used in energy market analysis
